## TodoMVC with *redux-mvc* and *redux-saga*

[Source](https://github.com/redux-mvc/core/tree/master/src/Examples/TodoMVC) &  [Live](https://redux-mvc.github.io/core/#/todo-mvc/)

### Instructions

- Chekout the source code

```
git clone https://github.com/redux-mvc/core.git
```

- Install the dependencies

```
yarn install
```

- Run the dev mode

```
yarn dev
```

- Go to [http://localhost:3000/todo-mvc#/](http://localhost:3000/todo-mvc#/)

- Check how sagas is added to the architecture [sagas recipe](https://github.com/redux-mvc/core/blob/master/src/Examples/App/utils.js)
